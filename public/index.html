Anthropic Models Manager

Centralized model configuration API

Why This Matters

Instead of hardcoding Claude model versions in 10+ production apps, this API provides a single source of truth. Each app makes a live API call to fetch the current model. When Anthropic releases a new model, update it once here and all apps instantly use the new version on their next API call—no redeployment needed.

Update Once Change model version in one place
Live Updates All 10+ apps get the new model instantly
Version Control Track which model is live at any time
